# 图片压缩工具

这是一个简单易用的在线图片压缩工具，具有苹果风格的界面设计。用户可以上传图片，预览压缩效果，并下载压缩后的图片。

## 功能特点

- 支持PNG、JPG等常见图片格式
- 实时预览压缩前后的图片效果
- 显示压缩前后的文件大小
- 可调节压缩比例
- 支持压缩后图片下载
- 优雅的苹果风格界面设计

## 技术实现

- 使用原生HTML5和CSS3开发
- 使用FileReader API读取图片
- 使用Canvas API进行图片压缩
- 采用Flexbox布局
- 响应式设计，支持各种设备
